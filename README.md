# Script de enviar mensagens no WhatsApp 

Script para abrir múltiplos chats do WhatsApp Web com mensagem pré-preenchida.

## Como usar

1. Copie `dados_exemplo.py` para `dados.py`  
   `cp dados_exemplo.py dados.py`

2. Edite `dados.py` com os nomes e links `wa.me` reais.

3. Execute: `python3 enviar_whatsaapp.py`

4. Faça login no WhatsApp Web e pressione ENTER.

5. Clique em "Enviar" em cada aba aberta.

## Aviso

- Os dados reais não são versionados (arquivo `dados.py` está no `.gitignore`).
- Use com responsabilidade, evitando spam.
